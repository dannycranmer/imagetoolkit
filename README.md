# ImageToolkit

Free, private image tools. Compress, resize, convert, crop, and watermark images entirely in your browser — your images never leave your machine.

**Live:** https://dannycranmer.github.io/imagetoolkit/

## Tools

- **Compress** — reduce image file size
- **Resize** — scale images to exact dimensions
- **Convert** — change between PNG, JPEG, WebP formats
- **Crop** — trim images to a specific area
- **Watermark** — add text or image watermarks

## Why?

Online image tools upload your photos to their servers for processing. ImageToolkit does everything client-side using the Canvas API. Your images stay on your device — no uploads, no tracking, no ads.

## Tech

Static HTML/CSS/JS using the browser Canvas API. Zero dependencies, zero server calls. Deployed on GitHub Pages.
